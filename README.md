# A4P Platform V6

V6 ajoute :
- historique des tests
- comparaison avant / après
- progression saison
- comparaison joueurs
- comparaison équipes
- synthèse staff avancée

## Déploiement GitHub Pages
1. Créer un nouveau dépôt GitHub, par exemple `A4P-Platform-V6`.
2. Envoyer tout le contenu de ce dossier à la racine du dépôt.
3. Activer GitHub Pages depuis la branche `main` et le dossier `/root`.
4. Ouvrir `index.html`.

## Pages principales
- `index.html` : vue club
- `equipe.html?team=u18` : vue équipe
- `joueur.html?player=u18-01` : vue joueur
- `comparaison.html` : comparaisons avancées

## Données
Le fichier `data/club-data.json` contient les données de démonstration. Il peut être remplacé par tes propres exports.
